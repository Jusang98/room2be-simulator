# 🛋️ Room2Be – 3D Furniture Layout Simulator

This project is a minimal prototype for **Room2Be**, a web-based 3D simulator that allows users to drag and arrange furniture in a virtual room. It's built to test and experiment with Three.js, object interactions, and room configurations using basic geometries.

> 🚧 MVP testing repo – not production-ready yet.
